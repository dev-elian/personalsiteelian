Thanos Whacker VR

"Thanos Whacker" is an application that consists of the Whac-a-Mole video game brought into virtual reality, using various resources to make the game more entertaining for the user.
The complete game is set in a game room according to the game's theme.

GitHub:https://github.com/dev-elian/VR-Whac-a-mole
Genre: Casual
Engine: Unity 2020
Team Size: Personal project
Duration: 2 Weeks
Platform: Oculus Quest, Oculus Rift
Roles and Responsibilities:
Game Designer: I designed the game mechanics and the basic user experience in a game room.
Programmer: I programmed the complete functionality (game flow, scoring system, visual effects with particle system, power-ups, overall user experience).
3D Modeler: I modeled and textured the game room, the game machine, and the mallets.
QA Tester: I conducted functionality and performance tests with Oculus Quest II using the Oculus OVR Profiler.

Programmer: All functionality using MVVM and connection to a WebAPI with .NET and Azure.
QA Tester: Quality testing.

CurioAdolescente
"CurioAdolescente" is an educational video game developed by Ancestral Gods for Save The Children (Bolivia) with the main goal of strengthening self-learning among adolescents for their personal development.
